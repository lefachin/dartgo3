# dartgo3
An example of a web form using Go, Angular Dart2, and MongoDB
# Go dependencies
<code>go get gopkg.in/mgo.v2</code>
# Compiling and Running
<code>
cd app<br>
pub build<br>
cd ..<br>
go build<br>
</code>
... then run the resulting executable
Lastly, point your browser to <em>http://localhost:8080</em>
